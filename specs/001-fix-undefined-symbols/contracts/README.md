# Robot Plugin API Contracts

This directory contains API contracts for the robot plugin system.

## Files

- **robot-plugin-api.md**: Complete API contract for robot plugins, including:
  - Plugin requirements
  - Robot base class API
  - Logger API
  - Trigonometry utilities
  - Type definitions and constants
  - Error handling
  - Best practices
  - Example templates

## Usage

These contracts define the interface that all robot plugins must implement. Use this documentation to understand:
- How to create a valid robot plugin
- What methods are available in the Robot API
- How to use logging and navigation utilities
- Common pitfalls and how to avoid them
